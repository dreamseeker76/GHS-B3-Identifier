# ⚗️ GHS B3 Identifier

Aplikasi identifikasi simbol bahaya bahan kimia berbasis **Globally Harmonized System (GHS)**.

## Fitur
- Identifikasi 9 simbol GHS lengkap
- Panduan Alat Pelindung Diri (APD)
- Prosedur tindakan darurat
- Sesuai regulasi Indonesia (PP 22/2021, PermenLHK P.12/2020)

## Cara Menjalankan Lokal
pip install streamlit
streamlit run app_ghs.py
